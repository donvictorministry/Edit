How to Use Code Repository

      
    

    

      
1. Login — Enter your Key to unlock the repository.

      
2. Explorer — Browse all projects and files. Works fully offline from cached data.

      
3. New Project — Creates a folder cloud storage (requires network).

      
4. New File — Opens the Code Editor for a new file in any project.

      
5. Code Editor — Full-screen editor. Snippets, Undo, Redo, Search, Line Numbers, Zoom, Wrap, Contrast toggle.

      
6. View Code — Opens the viewer with line numbers and copy button.

      
7. Live Preview — Renders .html files in a live preview...works offline.

      
8. Save Device — Downloads your code as a real file to your device. 100% offline.

      
9. Push Drive — Sends code to Drive via servers (requires network + admin key).

      
10. Sync Drive — Manually refreshes the project tree from Drive. Only time network is used.

      
11. Offline Drafts — Open the three-dot menu to view, load, and manage all local drafts.
